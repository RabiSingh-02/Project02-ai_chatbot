# Project02-ai_chatbot
AI-powered chatbot web application built with Node.js and Express, integrating Generative AI (LLM) via OpenRouter API to deliver real-time conversational responses through a responsive UI.


🤖 AI Chatbot Web Application

This project is a full-stack chatbot web app powered by AI. It delivers real-time responses using a Large Language Model (LLM) through the OpenRouter API, making conversations feel smooth and interactive.

⚙️ Features
💬 Chat with the AI in real time
⚡ Quick and efficient responses powered by an LLM
🎨 Simple, clean, and responsive interface built with Bootstrap
🔄 Seamless async communication using the Fetch API
🔐 Keeps API keys safe with environment variables
📦 Organized and modular backend using Express.js
🛠️ Tech Stack

Frontend:

* HTML5
* CSS3
* Bootstrap
* JavaScript (ES6)

Backend:

* Node.js
* Express.js
* EJS (templating engine)

API & Tools:

* OpenRouter API for AI responses
* Axios for handling HTTP requests
* dotenv for managing environment variables
